# MY PANCHAYAT

A smart AI powered gram panchayat management system.

## Running the application

To Run the container:
```bash
docker compose up -d
```
To Enable Hot Reloading (Watches for file changes & syncs them inside the container)
```bash
docker compose watch -d
```
